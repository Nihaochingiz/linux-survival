ls -l

 "ls" with the "long listing" option

 Выдает список файлов
 Первая строка
 Если "-", значит это файл

 Если "d", значит это директория, пример
 -rw-r--r--

Расшифровка
 -rw-r--r--
 r значит ты можешь читать файл
 w значить ты можешь писать и изменять файл
 x значит ты можешь исполнять файл, данное разрешение выдается исключительно Если
 файл является программой

  -rw-r--r--
  -rw-

  chmod
  command you  to change the security permissions
  Команда для изменения разрешения у пользователя

  Команда для того чтобы дать права  (other(world)) для выполнения
  программы
  chmod o
  + это если мы добавляем разрешение
  
  chmod o+

  Также в конце нужно написать файл, к которому мы добавляем
  разрешения

  chmod o+x gorillas

  Вы также можете изменить несколько разрешений одновременно.

  chmod ugo-rwx gorillas


  chmod g+w gorillaz
  дать write разрешение для группы к файлу "gorillaz"

  chmod: это команда в Linux, используемая для изменения разрешений доступа к файлам и директориям.
  g+w: это опция команды chmod, которая указывает, что мы хотим добавить разрешение на запись (write) для группы (group).
gorillaz: это имя файла или директории, для которого мы хотим изменить разрешения.

chmod g+w *
дать группе право на write во всей директории
